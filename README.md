# Continous Token

## Smart Contracts, Unittests and Infrastructure.

### Installation

#### Prerequisites

 * Python 3.6
 * [pip](https://pip.pypa.io/en/stable/)

#### Setup

 * pip install -r requirements.txt

#### Usage

 * populus compile
 * pytest
 * populus deploy
